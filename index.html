Restaurant Management Project

Developed a Restaurant Management System website that allows users to view real-time restaurant status (open/closed), check opening hours, and access a brief description of the restaurant. Authorized users can edit restaurant information, and visitors can easily subscribe to a newsletter via email to receive updates.

PowerPlay: Kinetic Energy Harvesting System Through Interactive Gaming

This project integrates manufacturing, kinematics, and instrumentation into a game platform where players' movements control gameplay. Shifting the plate left, for example, moves a car left in the game. Player motion powers a pulley system connected to a DC motor, generating electricity for an LCD with RPM feedback, while impacts from plate movement generate energy for LEDs via piezoelectric materials, ensuring interactive, energy-efficient feedback

Dual Sensor Alarm

This project focuses on creating a dual sensor alarm system using both heat and light detectors. The alarm activates when either sensor detects unusual changes, offering a more reliable and responsive safety system by monitoring temperature and light conditions simultaneously.


Car Transmission Design

This project aims to design a car transmission system that utilizes a finite state machine (FSM) to manage driving modes and gear selection. We implemented the design in Quartus software and subsequently developed the hardware to validate its functionality in a real-world setting.

ACMS: Advanced Class Management System

The project involves designing an attendance system using RFID cards and biometric fingerprint scanning, with data stored in a centralized database and accessible via an HTML interface. Additionally, a room booking system allows users to reserve rooms and monitor occupancy using motion sensors, ensuring efficient management of resources.
